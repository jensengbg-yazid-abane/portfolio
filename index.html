HTML

HyperText Markup Language, är en öppen standard språk som ägs av ingen och man behöver inte betala för en licens. Den används för att bygga webbsidor, man kan placera in t.ex. texter, tabeller, bilder, videor eller audiofiler mm. För att kunna skriva HTML behöver man ha en text editor och en webbläsare för att testa hur sidan ser ut.

CSS

Cascading Style Sheets, är stilmallar som tillåter både skaparen och användaren till att ändra stilmallar. Man kan justera t.ex. färg, teckensnitt, backrunder mm. CSS har gjort programmeringen enklare då har kodningen blivit enklare att läsa och underhålla genom att ha sin CSS kod direkt i HTML dokumentet eller genom att skapa ett CSS dokument som kopplas till HTML och som är den mest praktiska sättet.

JavaScript

JS, är ett scriptspråk som skrivs direkt i HTML oftast längst ner i dokumentet och gör att man kan skapa mer avancerade interaktiva sidor, med hjälp av JS kan man t.ex. skapa popup-fönster, rollover, dropdown menyer mm.

Element

är en hel kodsnutt från starttag till sluttag och ska innehålla content inuti, t.ex. html innehåll /html , body innehåll /body p innehåll /p.

CSS-ruleset

består av en selector och en declaration som i sin tur kan innehålla flera propertys och values, t.ex. h1 {color: black; font-siza: 11px;}.
